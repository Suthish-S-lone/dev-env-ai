# Dev-Env-AI 🚀  
AI-powered Development Environment Generator

## Problem Statement
Setting up development environments is time-consuming and error-prone.  
Dev-Env-AI automates environment creation using AI and containers.

## Solution
Users select:
- Programming language
- Frameworks / libraries

The system automatically generates:
- Dockerfile
- devcontainer.json
- Dependency list

## Architecture
- Electron.js (Desktop App Shell)
- Node.js backend
- AI model for environment generation
- Docker-based isolation

## Features
- System setup mode (coming soon)
- Container-based setup (implemented)
- AI-generated environment configuration
- Secure API key handling via environment variables

## Tech Stack
- Electron
- Node.js
- Docker
- AI APIs (Gemini / Groq / DeepSeek – pluggable)
- HTML / CSS / JavaScript (frontend)

## Security
- No hardcoded API keys
- Environment variable–based secrets
